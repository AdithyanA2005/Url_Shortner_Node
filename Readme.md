#### HOW TO USE

> Clone the repository
> ``` bash
> git clone https://github.com/AdithyanA2005/Url_Shortner_Node.git
> ```

> Install Dependencies
> ``` bash
> npm install
> ```

>
> NOTE: You want to have mongodb in working state
> 

> Start The server
> ``` bash
> npm start
> ```
